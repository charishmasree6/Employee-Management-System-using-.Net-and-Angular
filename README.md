# Employee Management System

## Tech Stack
- Backend: ASP.NET Core Web API (.NET)
- Frontend: Angular
- Database: SQL Server
- Tools: Visual Studio, VS Code, Postman

## Project Description
This is a full-stack Employee Management System developed using ASP.NET Core and Angular.  
The application allows managing employee records through CRUD operations.

## Features
- Add new employee
- View employee list
- Update employee details
- Delete employee records

## Backend Details
- RESTful APIs built using ASP.NET Core
- Entity Framework Core for database operations
- SQL Server as database

## Frontend Details
- Angular components for UI
- Services for API integration
- Forms for add/update operations

## Learning Outcome
- Frontend and backend integration
- REST API development
- CRUD operations
- Working with Angular and .NET together
